# Silhouette dataset

`countries.js` is a readable compiled data file loaded by the quiz at runtime.

Because the file combines Natural Earth public-domain material and GSHHG-derived LGPL material without retained per-entry provenance, the entire file is distributed under `LGPL-3.0-or-later` as a conservative licensing measure.

The file itself is the preferred form for modification. See the repository-level `LICENSE`, `THIRD_PARTY_NOTICES.md`, and `LICENSES/` directory.
